Esse programa tem seu Makefile preparado para funcionamento com uma
base de dados PostgreSQL local, instalada na máquina virtual usada
para o desenvolvimento deste trabalho. Sua execução não é garantida 
em outros tipos de bases de dados sem adaptações (porém pequenas)
no código fonte, e também não é garantida em bases de dados que
estejam rodando versão inferior a 9.5 do PostgreSQL.

A equipe.
